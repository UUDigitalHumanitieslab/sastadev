
SASTA en SASTAdev
=================

A distinction must be made between SASTA and SASTAdev. We will describe their characteristics below. SASTA and SASTAdev are both on github repository https://github.com/UUDigitalHumanitieslab/sastadev .

SASTA
-----

SASTA is the web application for public use. It is maintained by the DH Lab (Jelte van Boheemen). Its URL is https://sasta.hum.uu.nl 
It takes as input a word document or a CHAT file, or a (corrected) annotation file, and provides as output an annotation file, and a method-specific form file.
